# Common-Commands

git 命令
--------

查看是否有远程关联仓库
`git remote -v ` 

添加远程关联仓库
```git remote add origin [git address]```

删除远程关联仓库
`git remote rm origin`

上传代码
`git push origin master`

拉取代码
`git pull origin master`

tmux 命令
---------

新建会话
`tmux new -s [name]`

连接会话
`tmux a -t [name]`

pdb命令
-------------

打印 `pp`

下一步执行 `n`

步进 's'

列出 `l`

继续执行 `c`

返回 `r`
